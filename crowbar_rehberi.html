CROWBAR YAZILIMI KULLANIM REHBERİ

Crowbar Goldsrc ve Source grafik motorları için decompile, compile işlemlerini yapma , model görüntüleme vb. işlemleri yapabileceğiniz ücretsiz bir yazılımdır. Buradan indirebilirsiniz.

Bu rehberde Crowbar yazılımının kullanımı anlatılacaktır.

Set Up Games

Crowbar programında herhangi bir oyun için derleyeci, görüntüleyici ve paketleyici gibi özelliklere erişebilmek için belirli araçlara gereksinim vardır, crowbar otomatik olarak bu araçların yerini bulamaz, bu yüzden bunların yerini belirtmek gerekir, varsayılan olarak steam kütüphanesi adresi gösterilir

The Preview, Pack, Compile, ve View kısımları her biri Set Up Games butonuna sahiptir. Her biri aynı oyun listesine sahip iken her kısımda farkılı ayar değerleri bulunur

Oyun Seçme

En üstte hangi oyun için ayarlama yapacağınızı seçin.

Yeni Oyun Ekleme

Add butonuna tıklayarak ayar yapmak üzere yeni bir oyun ekleyin

Oyun Klonlama

Clone butonuna tıklayarak seçili oyunun aynı ayarlarına sahip yeni bir oyun ayarı oluşturun

"Create "models" folder tree from this game's VPKs" Butonu

Bir çok model göstericisi (HLMVler) .vpk dosyaları içinde bulunan modelleri bu dosya oyun dosyalarının bulunduğu dizinde değilse modeli gösteremez.Oyun içinde bulunan tüm .vpk dosyalarında bulunan "models" klasörünü okuması ve bu klasörleri oyun dosyaları içinde oluşturması için bu butona tıklayın.Bu şekilde sadece klasörler oluşur dosyalar ise oluşturulmaz

Steam Kütüphanesi Klasörleri

Steam oyunlarınızın başka bir dizinde olması durumunda Preview ve View kısımlarının "Run Game" özelliğini kullanabilmeniz için bu dizini belirtmeniz gerekmektedir

Add Macro butonuna tıklayarak yeni makro ekleyebilir, Browse butonuna tıklayarak da yeni dizin adresini belirtebilirsiniz.

Eklenen makroya fare ile sağ tıklandığında:

  • Set macro in selected game setup(s): seçili olan oyun ya da oyunları mevcut makronun adresinde olduğu ayarını yapar.
  • Clear macro in selected game setup(s): seçili olan oyun ya da oyunlardan mevcut makronun adresinde olduğu ayarını kaldırır.
  • Change to this macro in selected game setup(s): seçili olan oyun ya da oyunlar başka bir makronun adresine ayarlıysa mevcut makronun adresinde değiştirirlir.
  • Set macro in all game setups: Tüm oyunları makronun adresinde olduğu ayarını yapar.
  • Clear macro in all game setups: Makronun adresine ayarlı tüm oyunları makrodan çıkartır.
  • Change to this macro in all game setups: Diğer makroların adresine ayarlı tüm oyunları mevcut makronun adresine ayarlar.

Bu ayarlamalar oyun dosyalarının yerini değiştirmez, Crowbar'a oyun dosyalarını hangi dizinde bulacağını söyler.

Download

Steam Workshop üzerinden herhangi bir oyun için dosya indirmenizi sağlar. Dosyanın bulunduğu web adresi veya dosyanın IDsini "Item ID or link" kısmına yazmanız gerekiyor. "Output to" kısmında ise dosyaların hangi dizine indirileceği belirtiliyor. Daha sonra Download butonuna tıklayarak indirme işlemini başlatılıyor.

"Item ID or link" kısmına yazmak yerine kopyaladığınız ID ya da web adresini yapıştırabilirsiniz.

Crowbar ile sadece sahip olduğunuz oyunların workshop dosyaları indirilebilir.

Open butonuna tıkladığınızda ise girmiş olduğunuz ID veya web adresinin workshop sayfası açılır.

Output File Name Options

Bu kısımdaki seçenekler dosyanız indirildiğinde isminin nasıl gözükeceğini belirler. Example output file name" kısmında örnek dosya adının nasıl olacağını görebilirsiniz.

"Convert to expected file or folder" seçeneği indirilen Garry's Mod dosyasını GMA dosyasına çevirir.

Unpack

Crowbar herhangi Valve VPK arşivini , Garry's Mod GMA, Vampire: The Masquerade - Bloodlines VPK, Tactical Intervention FPX ve Fairy Tale Busters APK paketlerini çıkartabilir

Arşiv seçme

Sadece 1 adet arşiv çıkarmak istiyorsanız "Packages:" yazısının sağında "File" seçeneğini seçip, browse kısmından da arşiv dosyasını seçerek;

Birden fazla arşiv çıkarmak istiyorsanız "Folder" seçeneğini, seçeceğiniz dizinin alt dizinindeki arşivlerin de dahil edilmesi için "Folder and subfolders" seçeneğini seçip, arşivlerin bulunduğu dizini seçerek arşiv kaynağını belirtebilirsiniz

Arşiv çıktısı yeri belirlemesi

Same folder seçeneği arşivin(ler)in bulunduğu aynı dizine,

Subfolder seçeneği ile arşivlerin bulunduğu aynı dizine, ayrı bir klasöre,

Work folder seçeneği ile sizin belirteceğiniz bir klasöre çıkarılmasını ayarlayabilirsiniz. Yanındaki boşluğa adresi yazabilir, yapıştırabilir ya da Browse butonu ile istediğiniz dizini seçebilirsiniz.

Arşiv içinde gezinme

"Contents of package" kısmında üstteki kutuda arşiv içinde bulunduğunuz yerin adresi, sol kutuda klasörler, sağ tarafta da seçili olan klasörün içeriği gözükmektedir

Preview

Bu kısım model hakkında basit bilgi almak içindir.Buradan her oyunun kendi model göstericisine, harita yapıcısına veya oyunun kendisine erişebilir.

MDL file kısmındaki boşluğa incelemek istediğiniz modelin bilgisayarınızdaki adresini yazarak ya da browse tuşundan arayarak seçebilirsiniz.

Info

Model seçtikten sonra Info kısmında model hakkındaki bilgileri alabilirsiniz. Bu kısımdaki bilgiler hata aramada yardımcı olabilir, örnek olarak sıklıkla karşılaşılan eksik doku sorunu için hangi doku dosyalarının gerektiğini öğrenip hatalı model ile karşılaştırma yapılabilir.

Game that has the model viewer

Farkılı model göstericilerin farklı oyun ve oyun motorları için farklı özellikleri olabilir. Buradan hangi oyun model göstericisini kullanacağınızı seçin.

View, View as Replacement, Use in Decompile

Sol alt köşedeki "View" butonuna tıklamak seçilen modeli görmek için bir model gösterici açar.

Oyunda kullanılan aynı isimde bir model seçtiyseniz "View As Replacement" butonuna tıklayın. Bu durumda Crowbar yapması sıkıcı olan bir takım işleri arka planda halleder.

"Use in Decompile" butonu seçili modeli decompile işlemi yapmak için Decompile kısmına yönlendirir

Open Viewer, Open Mapper, Run Game

"Open Viewer" butonuna tıklamak herhangi bir model yüklemeden model göstericiyi açar.Crowbarı kapatmak model göstericileri de kapatır.

"Open Mapper" butonuna tıklamak harita yapıcısını (genellikle Hammer programı) açar.Crowbarı kapatmak aynı şekilde harita yapıcısını da kapatır.

"Run Game" butonu oyunu açar.

Decompile

MDL input

Buraya adresi yazarak, adresi yapıştırarak ya da "Browse" butonundan modeli seçerek, birden fazla model için modellerin bulunduğu klasörü seçerek hangi modellere decompile işlemi uygulanacağı seçilir. Goto butonu ile seçili dosya ya da klasörün bulunduğu klasör Windows Dosya Gezgini ile açılır.

Output to

Work folder seçeneğinde decompile işlemi sonunda oluşturulan dosyalar sizin belirteceğiniz bir klasörde depolanır. Yanındaki boşluğa adresi yazabilir, yapıştırabilir ya da Browse butonu ile istediğiniz dizini seçebilirsiniz.

Subfolder seçeneği ise dosyaları seçili modelin veya klasörun bulunduğu dizinde ayrı bir klasöre çıkartır

Goto butonu ile seçili dosya ya da klasörün bulunduğu klasör Windows Dosya Gezgini ile açılır.

Re-Create Files

Bu kısımda decompile işleminde hangi dosyaların çıkarılacağı ayarlanır.

QC file

Seçildiğinde model için .qc dosyası oluşturulur

Group into QCI files

Seçildiğinde komutlar gruplandırılır, hepsi tek bir .qc dosyasına yazılmaz.

Each $texturegroup skin-family on single line

Seçildiğinde modelin her bir doku dosyası için .qc dosyasında ayrı komut satırı yazılır

Only changed materials in $texturegroup lines

Unchecked = write all materials for each skin-family as stored in the MDL file

Checked = write only the materials than change between skin-families

Include $definebone lines (typical for view models)

Seçildiğinde .qc dosyasına $definebone komutlarını ekler.

Texture BMP files (GoldSource)

Seçildiğinde modelin doku dosyaları çıkartılır.

LOD mesh SMD files

Seçildiğinde Level of Detail model dosyaları çıkartılır.

Physics mesh SMD file

Seçildiğinde Physics model dosyaları çıkartılır.

Vertex animation VTA file (face flexes)

Seçildiğinde modelin yüz hareketleri için olan VTA dosyası çıkartılır.

Procedural bones VRD file

Seçildiğinde VRD dosyası çıkartılır (Source motoru için)

Bone animation SMD files

Seçildiğinde modelin animasyon dosyalarını da çıkartır.

Place in "anims" subfolder

Seçildiğinde animasyon dosyalarını diğerleriyle aynı klasöre çıkarmak yerine aynı dizinde "anims" adlı bir klasöre çıkartır.

"Use Defaults" button

Bu butona tıklamak ayarları varsayılana ayarlara geri çevirir.

Options

Folder for each model

Seçildiğinde her 3d modeli ayrı klasöre çıkartır.

Prefix mesh file names with model name

Seçildiğinde modelle aynı isme sahip 3d modellere karışıklık olmaması için ön ad ekler

Format for stricter importers

Seçildiğinde dosyalar eski araçlar tarafından kullanılan formatta yazılır. Kullanıcı tarafından okunabilirliği daha azdır.

Write log to a file

Seçildiğinde işlem sonunda işlem hakkında bir rapor dosyası oluşturur.

Decompile-info comments and files

Seçildiğinde dosya içine yazılan yorumları da çıkarır.

Compile

Crowbar kendi başına model derleyemez, derleme işlemini yapması için studiomdl derleyicisini çağırır.Bu yüzden derlemek istediğiniz modelin oyununun derleyicisine sahip olmanız gerekmektedir.

QC input

Buraya adresi yazarak, adresi yapıştırarak ya da "Browse" butonundan .qc dosyasını seçerek, birden fazla .qc dosyası için için bu dosyaların bulunduğu klasörü seçerek hangi dosyalar kullanılarak derleme işleminin yapılacağı seçilir. Goto butonu ile seçili dosya ya da klasörün bulunduğu klasör Windows Dosya Gezgini ile açılır.

Output to

Work folder seçeneğinde derleme(compile) işlemi sonunda oluşturulan model sizin belirteceğiniz bir klasörde depolanır. Yanındaki boşluğa adresi yazabilir, yapıştırabilir ya da Browse butonu ile istediğiniz dizini seçebilirsiniz.

Subfolder seçeneği ise modeli seçili .qc dosyasının bulunduğu dizinde ayrı bir klasöre çıkartır

Options

"Game that has the model compiler" kısmından hangi oyunun derleyicisi (genellikle studiomdl) seçilir.

Write log to a file

Seçildiğinde derleme işlemi sonunda bir rapor dosyası oluşturulur

Direct entry of command-line options (in case they are not included above)

Buraya üst kısımda belirtilmemiş derleyici komutları yazılır.

En alttaki kutucukta ise Crowbarın derleyiciye gönderdiği arka plan komutları gösterilir.

View

Pack
